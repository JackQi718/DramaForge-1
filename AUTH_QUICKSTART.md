## 使用流程

### 1. 配置 Supabase

在 `.env.local` 中添加：

```env
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key-here
```

### 2. 创建数据库表

在 Supabase SQL Editor 中运行 `lib/supabase/schema.sql`

### 3. 启动应用

```bash
npm run dev
```

### 4. 测试功能

1. 访问 `http://localhost:3000`
2. 自动跳转到 `/auth` 登录页
3. 注册新账号或登录
4. 登录成功后跳转到首页
5. 点击右上角头像查看用户菜单

## 常见问题

### Q: 注册后没有收到验证邮件？
A: 开发环境下，Supabase 会自动验证用户。生产环境需要配置邮件服务。

### Q: 如何修改用户信息？
A: 可以在 Supabase 仪表板的 Authentication → Users 中手动修改。

### Q: 如何重置密码？
A: 可以添加"忘记密码"功能，使用 Supabase 的密码重置 API。

### Q: 如何添加第三方登录（Google、GitHub）？
A: 在 Supabase 仪表板的 Authentication → Providers 中启用相应的提供商。
