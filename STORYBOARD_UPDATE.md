# 分镜表格更新说明

## 更新内容

### 1. 表格形式展示

分镜现在以专业的表格形式展示，包含以下列：

| 列名 | 说明 |
|------|------|
| 场景号 | 场景编号 |
| 场景描述 | 详细的场景描述（时间、地点、人物、动作） |
| 镜头运动 | 推镜、拉镜、跟镜、摇镜、固定镜头等 |
| 对话内容 | 角色对话（支持多行） |
| 出场角色 | 场景中的角色列表 |
| 画外音 | 旁白或画外音内容 |
| 画面色调 | 暖色调、冷色调、高对比度等 |
| 时长 | 预计拍摄时长 |
| 情绪 | 场景氛围情绪 |
| 视频拍摄提示词 | 完整的中文拍摄指导 |
| 操作 | 编辑/删除按钮 |

### 2. 中文视频提示词

AI 生成的视频提示词现在是中文格式，包含所有拍摄信息：

```
【镜头语言】推镜/拉镜/跟镜等
【画面内容】详细描述场景、人物、动作
【对话】角色对话内容
【画外音】画外音内容（如有）
【色调】画面色调描述
【氛围】情绪氛围
【时长】预计时长
```

### 3. 新增字段

- **画外音 (voiceOver)**: 旁白或画外音内容
- **画面色调 (colorTone)**: 色彩和光线描述

### 4. CSV 表格导出

导出功能现在生成 CSV 格式的表格文件，可以直接在 Excel 或 Google Sheets 中打开：

- 文件格式：`.csv`
- 编码：UTF-8 with BOM（确保中文正常显示）
- 包含完整的项目信息和所有分镜数据
- 表格结构清晰，便于打印和分享

### 5. 表格功能

- **横向滚动**：表格宽度较大时可以横向滚动
- **行内编辑**：点击编辑按钮可以直接在表格中修改
- **删除场景**：可以删除不需要的场景
- **自动编号**：删除场景后自动重新编号

## 使用方法

### 查看分镜表格

1. 在项目详情页面选择一个剧集
2. 点击"生成本集分镜"按钮
3. AI 会生成包含所有信息的分镜表格

### 编辑分镜

1. 点击场景行右侧的"编辑"按钮
2. 在表格中直接修改内容
3. 点击"保存"按钮保存修改

### 导出表格

1. 点击左侧边栏底部的"导出全部分镜"按钮
2. 系统会生成 CSV 文件并自动下载
3. 使用 Excel 或 Google Sheets 打开文件

## 提示词格式示例

```
【镜头语言】推镜，从远景推至中景
【画面内容】清晨的城市街道，阳光透过高楼洒下，主角李明穿着西装快步走在人行道上，手里拿着公文包
【对话】李明：今天一定要拿下这个项目！
【画外音】这是李明来到这座城市的第三年
【色调】暖色调，金色阳光，高对比度
【氛围】充满希望和决心
【时长】30秒
```

## 技术细节

- 表格组件：`components/storyboard-table-view.tsx`
- 数据类型：`lib/types.ts` 中的 `StoryboardEntry`
- API 路由：`app/api/generate/route.ts`
- 导出功能：CSV 格式，UTF-8 编码

## 注意事项

- 表格较宽，建议在大屏幕上使用
- CSV 文件使用 UTF-8 BOM 编码，确保 Excel 正确显示中文
- 视频提示词支持多行文本，便于详细描述
